computes an index-based for loop in parallel batches.

Parameters
beginfirst index of the loop.
sizethe loop runs in the range [begin, begin + size).
fa function (the 'loop body').
nThreadsthe number of threads to use; the default uses the number of cores in the machine; if nThreads = 0, all work will be done in the main thread.
nBatchesthe number of batches to create; the default (0) triggers a heuristic to automatically determine the number of batches.

Consider the following code:

std::vector<double> x(10);
for (size_t i = 0; i < x.size(); i++) {
x[i] = i;
}

The parallel equivalent is

parallelFor(0, 10, [&] (size_t i) {
x[i] = i;
});

The function sets up a ThreadPool object to do the scheduling. If you want to run multiple parallel for loops, consider creating a ThreadPool yourself and using ThreadPool::forEach().

Caution: if the iterations are not independent from another, the tasks need to be synchronized manually (e.g., using mutexes).

Parameters

nThreadsthe number of threads to use; the default uses the number of cores in the machine; if nThreads = 0, all work will be done in the main thread.
